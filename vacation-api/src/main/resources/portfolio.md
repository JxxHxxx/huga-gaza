결재 - 휴가 관리 서비스


기술 스택

```
jdk 17
spring-boot, batch, integration
databasae - mysql
data access - jdbc, jpa, mybatis
test - junit5
```

모듈

```
messaging : 결재 서버(타 서버)와 메시지 통신을 위한 시스템
batch : 휴가 데이터와 관련된 배치잡 실행을 위한 시스템
api : 휴가 HTTP API 서버
결재 서버
```

주요 기능

``` 
1. 휴가(반차) 신청
 : 신청 -> 결재 문서 기안 -> 결재선 지정 -> 상신

2. 요청에 대한 승인, 반려
  
3. 휴가 상태 관련 배치잡
  : 휴가 시작일에 맞춰 휴가상태 변경
  : 휴가 종료일에 맞춰 휴가상태 변경 및 잔여 연차 차감

4. 결재 문서 양식 추상화 
   : 휴가신청서 이외도 관리자 페이지에서 양식에 필요한 정보 입력 시 이에 맞게 결재 문서 생성 

5. 메시지 처리
   : 특정 이벤트 발생 시 메시지 생성, 메시지 서버에서 생성된 메시지 처리 (폴링)
   : 메시지 처리 실패 시 재처리 API

7. 로그 정책
  : HTTP API 접근 로그(API 서버의 경우), WAS 전체 로그로 분리
  : 1일 단위로 gzip 압축하여 보관
```
